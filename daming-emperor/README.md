# 🏯 重返大明 - 历代皇帝模拟器

从朱元璋到朱瞻基，体验不同大明皇帝的治国之路。一个高自由度的 AI 驱动文字冒险游戏。

## 🎮 游戏特色

- **五位皇帝任选**：
  - 👴 **朱元璋**（洪武）：铁血开国，猜忌心重
  - 👶 **朱允炆**（建文）：仁弱君子，削藩失败
  - ⚔️ **朱棣**（永乐）：雄才大略，靖难夺位
  - 🧓 **朱高炽**（洪熙）：仁厚守成，体弱多病
  - 🎨 **朱瞻基**（宣德）：太平天子，促织皇帝

- **朝代专属系统**：
  - 每位皇帝有独特的朝臣班子（如朱元璋有刘伯温、徐达）
  - 每位皇帝有专属历史事件（如朱允炆的靖难之役、朱棣的郑和下西洋）
  - 不同的初始属性和游戏难度

- **AI 驱动对话**：
  - 所有朝臣由 AI 实时生成独特对话
  - 不同性格、不同忠诚度的大臣会有不同反应
  - 太监总管的幽默吐槽

- **历史还原**：
  - 亲历历史大事件
  - 批阅专属奏折
  - 微服私访体验不同皇帝的民间故事

## 🚀 快速开始

### 1. 配置 API Key

编辑 `.streamlit/secrets.toml`，填入您的 KimiCode API Key：

```toml
[kimicode]
api_key = "sk-your-api-key-here"
```

### 2. 安装依赖

```bash
cd daming-emperor
uv sync
```

### 3. 启动游戏

```bash
uv run streamlit run streamlit_app.py
```

### 4. 在 Streamlit Cloud 部署

1. 将代码推送到 GitHub
2. 在 [Streamlit Cloud](https://streamlit.io/cloud) 部署
3. 在 Secrets 中添加：

```toml
[kimicode]
api_key = "your-api-key"
```

## 🎯 玩法说明

### 选择皇帝

游戏开始时选择你要扮演的皇帝，每位皇帝有：
- 不同的初始属性（朱高炽健康低，朱棣兵力高）
- 不同的专属朝臣（朱元璋有开国功臣，朱瞻基有三杨）
- 不同的专属事件（朱允炆要面对靖难之役，朱棣要北征蒙古）

### 核心玩法

- **💬 自由对话**：在底部输入框直接发号施令
- **👥 召见大臣**：左侧显示该皇帝专属的朝臣班底
- **📝 批阅奏折**：处理该朝代专属的历史奏折
- **🎲 微服私访**：体验该皇帝专属的微服私访事件
- **🏰 回宫休息**：恢复龙体健康（朱高炽休息效果特别好）

### 属性系统

| 属性 | 作用 | 危险线 |
|------|------|--------|
| 👑 皇权 | 朝廷掌控力 | ≤0 被篡位/软禁 |
| ❤️ 民心 | 百姓拥护度 | ≤0 被起义 |
| 💰 国库 | 银两储备 | 过低无法发军饷 |
| ⚔️ 兵力 | 军队战力 | 过低无法御敌 |
| 💪 健康 | 龙体安康 | ≤0 驾崩 |

### 特殊机制

- **朱允炆**：靖难之役是宿命，能撑过4年就是胜利
- **朱高炽**：历史上只活了10个月，游戏中撑过5年是奇迹
- **朱棣**：健康恢复快，但连年征战会消耗国库
